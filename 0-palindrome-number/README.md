## Palindrome Number

Given an integer `x`, return `true` if `x` is a palindrome, and `false` otherwise.
Palindrome: a word, number, phrase, or other sequence of symbols that reads the same backwards as forwards.

###

##### Example 1:

Input: x = 121

Output: true

##### Example 2:

Input: x = 743

Output: false

##### Example 3:

Input: x = -181

Output: false
